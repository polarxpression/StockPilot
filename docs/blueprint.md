# **App Name**: StockPilot

## Core Features:

- Add/Edit Cartridge Types: Allow users to add new cartridge types with details such as name, model, and set reorder thresholds.
- Update Stock Levels: Enable users to manually update the current stock level for each cartridge type.
- Restock Report Generation: Generate a report listing all cartridges that are below their reorder threshold.
- Intelligent Alerting Tool: Leverage AI to predict cartridge usage trends based on historical data, and to identify potential stockouts before thresholds are reached. When generating the alerts, the AI tool decides what time of day it would be most helpful to see these alerts.
- CSV Export: Enable exporting the restock report as a CSV file for further analysis or ordering.
- Image Export: Enable exporting the restock report as an image file.
- PDF Export: Enable exporting the restock report as a PDF file.

## Style Guidelines:

- Primary color: Deep purple (#673AB7) for a professional and calm feel.
- Background color: Light purple (#F3E5F5) offering a clean and neutral backdrop.
- Accent color: Soft red (#FF7043) used exclusively for low stock alerts and critical actions, providing a clear visual cue.
- Font: 'Inter', a grotesque-style sans-serif, providing a modern, machined, objective, neutral look for both headers and body.
- Use simple, line-based icons for cartridge types and actions. Ensure icons are easily distinguishable and visually consistent with the overall design.
- Maintain a clean and structured layout with clear visual hierarchy. Use whitespace effectively to avoid clutter and improve readability.
